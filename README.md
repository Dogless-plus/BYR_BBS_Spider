
北邮人论坛爬虫【皮皮虾系统】

一.已实现功能  
1.实习招聘板块、今日十大板块爬取
2.使用文件数据库
3.邮件订阅列表
4.定时功能

二、待实现功能
1.更多的数据库接口
2.https或者sock5代理池
3.用户自定义的过滤
4.可视化及dashboard
5.更多论坛版块
6.分布式
7.搜索引擎
8.发布评论
9.数据挖掘


三、注意事项
1.目前支持gmail作为服务端，但是不支持作为客户端
